Generate Electricity Bill
−

Create a Java program with a class named ElectricityBill that takes user input for the following details: consumer number, consumer name, previous month reading, current month reading, and EBconnectiontype (either "domestic" or "commercial"). Implement a method called calculateBillAmount() to compute the bill based on units consumed using a slab-based tariff system.

For domestic connections:

  • First 100 units: Rs. 1 per unit.
  • 101-200 units: Rs. 2.50 per unit.
  • 201-500 units: Rs. 4 per unit.
  • Above 500 units: Rs. 6 per unit.

For commercial connections:

  • First 100 units: Rs. 2 per unit.
  • 101-200 units: Rs. 4.50 per unit.
  • 201-500 units: Rs. 6 per unit.
  • Above 500 units: Rs. 7 per unit.

Input Format:

  • The program should accept the following inputs in order:
Consumer Number (integer)
Consumer Name (String)
Previous Month Reading (integer)
Current Month Reading (integer)
EB Connection Type (String - "domestic" or "commercial")

Output Format:

  • The program should display the bill information in the following format:
Electricity Bill Details:
Consumer Number: [consumerNo]
Consumer Name: [consumerName]
Previous Month Reading: [previousMonthReading]
Current Month Reading: [currentMonthReading]
EB Connection Type: [ebConnectionType]
Bill Amount: Rs. [calculatedBillAmount]

Note:

  • Ensure the program validates that the current reading is not less than the previous reading.
  • The connection type should be case-insensitive (accept "Domestic", "DOMESTIC", etc.)
  • Display the bill amount with 2 decimal places for better readability.
